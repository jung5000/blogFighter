# BlogFighter

BlogFighter는 핵심 키워드를 입력하면 리서치부터 최종 HTML 미리보기까지 블로그 글 작성을 완벽하게 자동화하는 AI 에이전트 시스템입니다.

## 주요 기능

- **자동화된 웹 리서치**: 입력된 키워드를 바탕으로 블로그 주제를 기획하고, 웹에서 관련 정보를 수집하여 리서치 문서를 생성합니다.
- **플랫폼 맞춤형 글쓰기**: 네이버 블로그 및 티스토리 각각의 SEO 가이드와 문체 가이드에 맞추어 최적화된 포스트 초안을 작성합니다.
- **이미지 자동 생성**: 본문 문맥에 어울리는 이미지를 HTML+CSS로 디자인한 뒤, 자동으로 PNG로 캡처하여 문서 내 적절한 위치에 삽입합니다.
- **최종 조립**: 작성된 마크다운 초안을 검토용 `final.md` 파일과 브라우저에서 시각적으로 바로 확인할 수 있는 `final.html` 파일로 변환해 줍니다.

## 폴더 구조

```text
blogfighter/
├── agents/          # 서브 에이전트들의 작업 지시서 (리서처, 라이터, 이미지 메이커, 어셈블러)
├── guides/          # 에이전트가 참조하는 가이드 (스타일, 플랫폼별 SEO, 이미지 가이드)
├── output/          # 생성된 결과물이 저장되는 디렉토리 (주제별로 폴더 생성)
├── CLAUDE.md        # 메인 에이전트(오케스트레이터)용 시스템 가이드
└── README.md        # 프로젝트 설명서
```

## 워크플로우

BlogFighter는 총 4가지 단계를 거쳐 결과물을 생성합니다:

1. **Step 1 — 리서치**: 5가지 주제를 제안받고, 선택한 주제에 대해 웹 리서치를 수행합니다. (`output/[주제]/research.md` 생성)
2. **Step 2 — 글쓰기**: 리서치 결과와 플랫폼별 SEO 가이드를 바탕으로 초안을 작성합니다. (`output/[주제]/draft.md` 생성)
3. **Step 3 — 이미지 생성**: 초안 내 이미지 마커(`[IMAGE: ...]`)를 인식해 디자인을 생성 및 캡처 후 실제 경로로 치환합니다.
4. **Step 4 — 최종 조립**: 최종 마크다운 및 미리보기 가능한 HTML 파일로 빌드합니다. (`final.md`, `final.html` 생성)

## 커스터마이징

이 프로젝트는 다수의 에이전트가 순차적으로 동작하는 오케스트레이션 방식을 사용합니다. 
- 글쓰기 말투나 스타일을 바꾸고 싶다면 `guides/style-guide.md`를 수정하세요.
- 플랫폼별 SEO 로직을 변경하고 싶다면 `guides/naver-seo-guide.md`나 `guides/tistory-seo-guide.md`를 수정하세요.
- 각 단계별 에이전트의 작동 방식을 변경하고 싶다면 `agents/` 폴더 내의 지시서를 수정하면 됩니다.
