# 소네쏙 sonesoc 아이차 냉각 미니 선풍기 제품 명세서

## 📋 제품 기본 정보
* **브랜드:** 소네쏙 sonesoc
* **제품명:** 소네쏙 아이차 냉각 미니 선풍기
* **모델명:** SOC-C19

---

## ✨ 핵심 기능 (Key Features)
* **급속 냉각 패드:** 주변 공기 온도를 낮추는 것이 아닌, 피부에 직접 닿을 때 즉각적인 쿨링 효과를 제공하는 아이스 펠티어 소자 탑재
* **100단계 정밀 풍속:** 1단계부터 100단계까지 사용자가 원하는 대로 미세 바람 조절 가능
* **대용량 배터리:** 3000mAh 배터리 탑재 (2.5시간 완충 시 최대 8시간 작동)
* **강력한 성능:** 17,800RPM 총강력 BLDC 모터 적용
* **직관적인 LED 표시:** 배터리 잔량, 풍속 단계, 냉각 패드 작동 상태를 한눈에 확인 가능
* **초경량 & 초저소음:** 150g의 가벼운 무게와 39dB의 조용한 사용 환경
* **다용도 거치 (듀얼 거치):** 탁상용, 핸디용, 고리 겸용으로 상황에 맞게 사용 가능
